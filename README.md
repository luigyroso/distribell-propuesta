# Distribell — Propuesta Web

Propuesta de rediseño web para **Distribell - Quesos y Quesos** (desde 1987). Distribución institucional de alimentos en Bogotá.

**Live:** https://luigyroso.github.io/distribell-propuesta/

## Enfoque

- Distribución institucional (HoReCa, Retail, Domicilios)
- 5 categorías de producto (Quesos, Lácteos, Cárnicos, Derivados, Panadería)
- Puntos de venta y cobertura en Bogotá
- Diseño premium con scroll animations y tipografía editorial
